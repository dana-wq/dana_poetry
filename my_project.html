Module to implement the Fizz Buzz game.

my_project.fizz_buzz.checkio(num: int) → str¶

Returns ‘Fizz’ if the input is divisible by 3, ‘Buzz’ if divisible by 5, ‘Fizz Buzz’ if divisible by both, and the input number as a string otherwise.

my_project.pass6 module¶

Module to check the acceptability of passwords.

my_project.pass6.is_acceptable_password(password: str) → bool¶

Check if the password is acceptable based on certain criteria. Criteria: - At least three unique characters (alphanumeric). - Not containing the word ‘password’. - If length is greater than 9, it’s acceptable. - If it’s not entirely numeric. - If length is greater than 6 and contains at least one digit.
